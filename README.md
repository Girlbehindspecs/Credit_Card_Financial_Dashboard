Credit Card Financial Dashboard 📊
📌 Project Overview

This project is a Credit Card Financial Dashboard built using Power BI to analyze credit card transactions and customer data. The dashboard provides insights into revenue trends, transaction volume, customer demographics, and credit card usage patterns.

The goal of this project is to help stakeholders understand financial performance and customer behavior through interactive visualizations and key metrics.

🎯 Project Objective

Analyze credit card transaction data and customer information.

Track key financial metrics such as revenue, transactions, and customer activity.

Identify spending patterns and trends.

Provide actionable insights through an interactive dashboard.

🗂️ Dataset

The project uses two main datasets:

Credit Card Transaction Data

Customer Information Data

These datasets include details like:

Transaction amount

Card category

Customer demographics

Revenue and interest

Transaction count

🛠️ Tools & Technologies Used

Excel – Data preparation

SQL / MySQL – Data storage and querying

Power BI – Data visualization and dashboard creation

Power Query – Data cleaning and transformation

DAX – Calculated measures and KPIs

⚙️ Project Workflow

Collected and prepared datasets in Excel.

Imported datasets into SQL database.

Connected SQL database with Power BI.

Performed data cleaning and transformation using Power Query.

Created relationships and measures using DAX.

Designed interactive dashboards with charts and KPIs.

Analyzed trends and generated insights.

📊 Dashboard Features

Total Revenue Analysis

Transaction Amount & Count

Customer Demographics

Card Category Performance

Weekly Revenue Trends

Geographic Analysis

Activation & Delinquency Rate

📈 Key Insights

Revenue trends across different weeks.

Transaction growth and customer activity.

Customer segments contributing most revenue.

Card categories generating highest transactions.

Regional contribution to overall revenue.
